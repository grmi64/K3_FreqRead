myCreate Help



myCreate allows one to control basic movements of the iCreate Robot using the mouse or keyboard on your Apple OS X 10.5 and higher system. 
First, select the correct serial device  from the pull down list.  If using Bluetooth, the correct device will probably be '/dev/tty.ElementSerial-ElementSe'.  Once the device is open then your iCreate can be controlled using the buttons on the application panel or via the arrow keys on your keyboard.  The Space bar corresponds to the 'Stop' button.  The buttons and keyboard are disabled until the open is successful.
Use the slider control to vary the speed at which the robot travels.  The distance traveled between each control activation is displayed as well as the total distance traveled since the port was opened.  To return the robot to an idle state simply quit the application.
